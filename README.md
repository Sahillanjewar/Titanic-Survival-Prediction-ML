# Titanic-Survival-Prediction-ML
Machine Learning project that predicts whether a passenger survived the Titanic disaster using classification algorithms and data analysis techniques in Python.

# Titanic Survival Prediction

## Project Overview
This project predicts whether a passenger survived the Titanic disaster using Machine Learning techniques. The dataset contains passenger information such as age, gender, ticket class, fare, and cabin.

The goal is to build a classification model that can predict survival based on these features.

---

## Dataset
The dataset includes the following features:

- PassengerId
- Pclass (Ticket Class)
- Name
- Sex
- Age
- SibSp (Number of siblings/spouses aboard)
- Parch (Number of parents/children aboard)
- Ticket
- Fare
- Cabin
- Embarked
- Survived (Target Variable)

---

## Exploratory Data Analysis

Key insights discovered:

- Women had a higher survival rate than men.
- Passengers in first class had higher survival chances.
- Younger passengers had better survival probability.

---

## Results

The best performing model achieved an accuracy of approximately **80–85%**.
